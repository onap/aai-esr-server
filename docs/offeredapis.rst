ESR server Component Offered APIs
==================================
The esr server module provides the following APIs.

* VIM register
* VIM query
* VIM update
* VIM Deletion
* VNFM register
* VNFM query
* VNFM update
* VNFM Deletion
* EMS register
* EMS query
* EMS update
* EMS Deletion
* thirdparty SDNC register
* thirdparty SDNC query
* thirdparty SDNC update
* thirdparty SDNC Deletion
